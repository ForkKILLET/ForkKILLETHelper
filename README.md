## 实体

### `AddictingBlock` 上瘾方块

触发后玩家进入上瘾状态，1.5 秒后死亡，再次触发可以重置死亡计时器。切板会清除上瘾状态。

触发方式：站在上面 和/或 攀爬侧面，可以触发的边显示为粉红色。

### `NoAddictingStrawberry` 不上瘾草莓

只要会话内进入过上瘾状态，不上瘾草莓就会飞走。（重新开始章节可以让草莓回来）
